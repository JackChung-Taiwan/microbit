一、什麼是 micro:bit？

micro:bit 是一塊小型程式學習板，可以用來學習程式設計、電子控制與創意發明。它雖然小小一片，卻像一座迷你實驗室，可以顯示圖案、偵測動作、感受光線，還能做出互動作品。

初學者可以使用 MakeCode 積木程式 控制 micro:bit，例如讓 LED 顯示愛心、按按鈕計分、搖晃時出現骰子點數，或利用光線感測器製作小夜燈。

二、micro:bit 可以用到哪些感測器？

光線感測

光線感測器

可以偵測環境亮不亮。適合做小夜燈、亮度提醒器。

動作感測

加速度感測器

可以偵測搖晃、傾斜、移動。適合做骰子、運動計步器。

方向感測

指南針感測器

可以偵測方向。適合做簡單指南針或方向挑戰遊戲。

溫度感測

溫度感測器

可以偵測目前溫度。適合做溫度提醒器。

按鈕輸入

A、B 按鈕

可以讓使用者輸入指令，例如按 A 加分，按 B 歸零。

LED 輸出

5 × 5 LED 燈

可以顯示圖案、數字、箭頭、文字，是 micro:bit 的小小表情板。

四、micro:bit 實作題

實作題 1：按鈕打招呼

任務：按下 A 按鈕時，LED 顯示「Hi」。

提示：使用「當按鈕 A 被按下」與「顯示字串」。

當按鈕 A 被按下 顯示字串 "Hi"

實作題 2：搖一搖骰子

任務：搖晃 micro:bit 時，隨機顯示 1 到 6 的數字。

使用感測器：加速度感測器。

當搖晃 設定 number 為 1 到 6 的隨機數 顯示 number

實作題 3：光線小夜燈

任務：如果光線太暗，就顯示月亮圖案；如果很亮，就顯示太陽圖案。

使用感測器：光線感測器。

重複無限次 如果 光線亮度 小於 80 顯示月亮圖案 否則 顯示太陽圖案

實作題 4：簡易溫度提醒器

任務：如果溫度超過 30 度，顯示驚嘆號；否則顯示笑臉。

使用感測器：溫度感測器。

重複無限次 如果 溫度 大於 30 顯示圖示：驚訝 否則 顯示圖示：笑臉

六、如何開啟 GitHub Pages？

  1. 進入這個 Repository 的 Settings。
  2. 左側找到 Pages。
  3. Source 選擇 Deploy from a branch。
  4. Branch 選擇 main，資料夾選 / root。
  5. 按下 Save，等待 GitHub 產生網址。
